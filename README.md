## Numbers

|---|---|---|---|---|
| Install size | **5 MB** | 400 MB | 300 MB | 400 MB |
| RAM usage | **8 MB** | 200-400 MB | 400-1000 MB | 500+ MB |

| Dependencies | **0** | Rust ecosystem | Electron + Node.js | Electron + Node.js |
| Runs on Remote Desktop | **Yes** | No | Yes | Yes |


## Features

- 6 language lexers — C/C++, Java, JavaScript, Python, Go, VLO
- Compiler integration — call external compilers directly from the editor
- SSH & remote file access
- 15+ custom native GUI widgets — built on Win32/GDI, no Qt, no MFC
- Custom memory allocator — no malloc, fully tracked allocations
- Event dispatch system
- Multi-tab editing
- Project file management
